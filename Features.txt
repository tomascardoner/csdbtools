FEATURES:
=========
- Connect to any OLEDB database
- Load Table Properties
- Compare Tables, Columns, etc. (in order and by DataTables, not TreeView)
- Check Primary Keys
- Check Table Constraints
- Check Tables and Columns that exists in Database 2 and not in Database 1
